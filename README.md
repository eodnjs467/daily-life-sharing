## 2. 화면 구성 📺
- 로그인 전
    - [ ] 로그인
    - [ ] 회원가입
    - [ ] 초기 페이지
- 로그인 후(서비스 이용화면)
    - (navigation)
      - [ ] 홈
      - [ ] 탐색하기
      - [ ] 알림
      - [ ] 쪽지
      - [ ] 프로필
      - [ ] 글쓰기
    - (main)
      - [ ] 글쓰기
      - [ ] 피드 (추천 | 팔로우)
- 컨텐츠 상세페이지
    - [ ] 피드 상세 페이지
- 검색페이지
    - [ ] 입력된 검색어 관련 컨텐츠 목록
- 설정 페이지
    - [ ] 회원정보 수정(비밀번호, 닉네임, 프로필)
